# GamerBoxd por Luís Henrique e Miguel Augusto

### Projeto desenvolvido como parte do minicurso de Desenvolvimento Web para a EPIC.

Gamerboxd é uma plataforma social para compartilhar seus gostos em jogos permitindo salvar suas reviews de jogos dinâmicamente.

# Funcionalidades

* Review de jogos

    No Gamerboxd você terá suas reviews salvas, para que outras pessoas possam ver.

* Buscas dinâmicas

    Busque por um jogo na barra de pesquisa.

# Técnologias utilizadas
  * HTML5: Estrutura da aplicação.
  * CSS3: Estilização e layout responsivo.
  * JavaScript: Lógica e interatividade.
  * LocalStorage: Armazenamento de dados no navegador (Review dos jogos).
